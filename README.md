# PPVC

PPVC (or Python Package Version Checker) is a simple command-line utility that queries for the available versions of a library on PYPI.

## Installation

~~~bash
pip install ppvc
~~~

## Usage

~~~bash
# Lists all versions
ppvc requests
# Shows only the latest version
ppvc requests --latest
~~~
